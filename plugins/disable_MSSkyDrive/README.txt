Disable Microsoft Sky Drive (One Drive)

ABOUT
Disable Microsoft Sky Drive (One Drive) file Synchronization

	
CONFIGURATION
The following options can be added to config.ini

	DISABLEMSSKYDRIVE (Boolean) - Enable or disable the disable_MSSkyDrive plugin
		Y	- Enable the Example Plugin (DEFAULT).
		N	- Disable the Example Plugin.


NOTE
Disabling this plugin does not undo the changes made in previous runs.
